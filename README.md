# High-Risk-Patient-Prediction-for-NovaGen
This project focuses on predicting high-risk patients using machine learning, with a strong emphasis on maximizing recall to ensure critical cases are not missed. Multiple classification models—including Logistic Regression, KNN, Gradient Boosting, Random Forest, and a Voting Classifier—were implemented and evaluated. Among them, Random Forest achieved the best performance with a recall of 95.8% and strong overall accuracy, making it the most reliable model for this healthcare-focused application. The project highlights the importance of aligning model evaluation metrics with real-world clinical priorities.
